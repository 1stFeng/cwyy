项目名称《宠物医院》

####### 请严格遵守命名规范 #######

1. 这个小项目是用来练手的 java 项目，会抽离出一些遇到的应用场景的统一解决方案

2. 项目采用 SpringMVC 框架，使用 Hibernate 实现 JPA 接口

3. 这只是一个单纯的后端项目，这里希望能够做到前后端分离，前端项目后续会补充进来，目测使用react

-- 大爷带你去战斗

############ 项目启动 #############

1. 项目需要 java 环境，具体包括 jdk1.8 tomcat8 mysql5.6 maven3.2

2. 将 cwyy-web 项目通过 maven 命令打成 war 包直接放到 tomcat 的 webapps 文件夹下，windows 环境下点击 lib 目录下的 start.bat 批处理命令即可

3. 也可以将项目导入到 eclipse idea 等工具通过工具打包 配置 tomcat 或者 jetty 服务器运行即可
